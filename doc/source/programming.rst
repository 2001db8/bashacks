Programming
===========

Session for the creation of facilitators for development in cli


bh_skel_c
---------
Generates on the standard  output ``C`` skeleton.

.. note::

    Usage

        ``bh_skel_c``

.. code-block:: bash 
    
    $ bh_skel_c  
    #include <stdio.h>

    int main(int argc, char *argv[]) {

        return 0;
    }
